# OOP-inl-mning
OOP inlämningsuppgift
Det här programmet heter IOTsystem och är skrivet i C++.
Det simulerar två sensorer – en för temperatur och en för luftfuktighet – som skapar slumpmässiga mätvärden.
Användaren kan i en enkel meny läsa nya värden, se alla mätningar och visa statistik som medelvärde, min- och maxvärde.
Syftet är att träna på grundläggande objektorienterad programmering med klasser, strukturer och menyer i C++.
